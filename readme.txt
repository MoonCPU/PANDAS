Programa feito para analisar dados de vendas de uma arquivo excel usando o módulo pandas do python.

1)O programa só foi feito para o arquivo excel "Vendas.csv" em específico, não funciona com nenhum outro arquivo!

2)Ao iniciar o servidor, uma página online vai ser aberto automaticamente. Essa página contêm os dashboards dos dados do arquivo excel.

3)Importante notar que o servidor faz o programa travar. Neste caso, é necessário manualmente desligar a execução do servidor indo para o console do compilador python e digitar CTRL+C. Por esse motivo, o programa não pôde ser convertido em um arquivo executável e só pode ser rodado em código.

4)Os botões ticket médio, faturamento, quantidade e produto vão gerar, respectivamente, um arquivo em formato .png para esses dados no diretório do código.


*O item 3 causa muitos problemas de compilação e lentidão, considerando remover essa função do programa.
